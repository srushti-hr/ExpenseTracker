# Expense Tracker

A modern mobile Expense Tracker built using **React Native** and **Expo**. The application helps users record, organize, and analyze their financial transactions through an intuitive interface, interactive visualizations, and a built-in Smart Spending Assistant.

## Features

- Add Income and Expense transactions
- Support for Current and Past transactions
- Date & Time Picker
- Category-wise transaction management
- Multiple payment sources/accounts
- Transaction History with search
- Transaction Details screen
- Total Balance, Income & Expense overview
- Interactive Insights dashboard
  - Monthly Overview
  - Category-wise Pie Chart
  - Day-wise Spending Graph
- Smart Spending Assistant
- Clean and modern UI
- Local data persistence using AsyncStorage

## Tech Stack

- React Native
- Expo
- React Navigation
- AsyncStorage
- react-native-chart-kit
- react-native-svg

## Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ExpenseTracker.git
```

Install dependencies

```bash
npm install
```

Start the project

```bash
npx expo start
```
